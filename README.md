# Fighter CHAR - 2D Fighting Game

A custom fighting game built in GameMaker Studio 2. Battle as one of 4 unique characters, execute combo chains, charge special meters, and transform into powerful forms.

## Features

- 4 playable characters: Fighter, Samurai, Cook, and Wind Ninja
- Combo chain system with light and heavy attacks
- 5-meter charge system for special moves and transformations
- CPU AI with multiple difficulty levels
- Dynamic camera that zooms based on player distance
- Player vs Player, Player vs CPU, and CPU vs CPU modes

## Controls

| Action | Player 1 | Player 2 |
|--------|----------|----------|
| Move | WASD | Arrow Keys |
| Attack 1 | E | K |
| Attack 2 | R | L |
| Attack 3 | F | M |
| Special 1 | 1 | O |
| Special 2 | 2 | U |
| Dash | X | Space |
| Charge | Z | P |

## Characters

- **Fighter**: Balanced close-range brawler with strong combos
- **Samurai**: Heavy-hitting sword fighter with powerful specials
- **Cook**: Tricky movement and unpredictable attacks
- **Wind Ninja**: Fast and agile hit-and-run style

## Game Modes

- Player vs Player - Two human players
- Player vs CPU - Fight against AI
- CPU vs CPU - Watch AI battles

## How to Play

1. Select your character and game mode
2. Fill special meters by landing attacks or charging
3. Use special moves when meters reach 50%
4. Fill both meters to transform and access stronger attacks
5. Deplete opponent's health to win the round

## Development

Built with GameMaker Studio 2 using custom hitbox detection, frame-specific attack timing, and state machine character control.
